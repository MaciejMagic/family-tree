# Family tree app

Command-line tool for generating a family tree as a svg/pdf format graphic

---

Tools in use: Python, SQLite

Requirements: Python 3.10+, [Graphviz](https://www.graphviz.org/) for diagram rendering

## Usage

```
$ python3 app/main.py
```
